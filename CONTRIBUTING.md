# Contributing to Wedding OTT

Welcome to the Wedding OTT project team! We are a group of up to 10 members working together to build a premium streaming experience for weddings.

## Team Structure
- **Core Developers**: 10 Members
- **Platform**: Next.js, Cloudflare Workers, Oracle/AWS infrastructure

## Getting Started
1. Clone the repository.
2. Install dependencies: `npm install`.
3. Set up environment variables from `.env.example`.
4. Run the development server: `npm run dev`.

## Collaboration Workflow
1. **Branching**: Create a new branch for every feature or bug fix (`feature/your-feature` or `bugfix/issue-name`).
2. **Pull Requests**: Submit a PR for review before merging into `main`.
3. **Communication**: Please use the dedicated team channel for coordination.

## Code Standards
- Use TypeScript for all new code.
- Follow the existing project structure in `src/`.
- Ensure all API routes are properly documented.

## Adding Team Members
If you are the repository owner, you can add team members (up to 10 total) by:
1. Navigating to the repository on GitHub: `https://github.com/gnanicloud01/wedding14`.
2. Clicking **Settings** > **Collaborators**.
3. Clicking **Add people** and entering their GitHub username or email.
